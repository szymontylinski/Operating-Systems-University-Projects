# Operating Systems — University Projects in C++

This repository contains three projects developed in C++ for the **Operating Systems** course at the **Faculty of Information and Communication Technology**, **Wrocław University of Science and Technology**.

The projects focus on practical aspects of operating systems, including multithreading, process management, synchronization, parallel execution, and performance analysis.

## Projects

### 1. Two-Threaded Dijkstra Algorithm

The goal of this project was to implement and analyze a two-threaded version of Dijkstra’s shortest path algorithm.

The project focuses on:

* implementation of Dijkstra’s algorithm in C++,
* use of two threads to divide computational work,
* comparison of performance against a standard approach,
* analysis of execution time and efficiency.

### 2. Dining Philosophers Problem

This project implements the classic **Dining Philosophers Problem**, which is commonly used to demonstrate synchronization issues in operating systems.

The project covers:

* thread synchronization,
* shared resource management,
* avoiding deadlocks,
* modeling concurrent access to limited resources.

### 3. Parallel Electoral Process Simulation

The goal of this project was to create a system simulating an electoral process with multiple local commissions working in parallel and one central commission responsible for collecting and verifying results.

The simulation includes:

* local commissions implemented as separate processes,
* voters simulated as multiple threads inside each local commission,
* parallel vote processing,
* sending results from local commissions to the central commission,
* aggregation and verification of election results.

## Technologies

* C++
* Multithreading
* Processes
* Synchronization mechanisms
* Inter-process communication

## Purpose

The purpose of this repository is to present practical implementations of selected operating system concepts. Each project demonstrates a different aspect of concurrent or parallel programming, showing how processes and threads can be used to solve computational and synchronization problems.
